## Introduction
A telnet server made in [Elixir](https://elixir-lang.org/) for practice. It is my first server ever.

## Quickstart
### Server
```sh
$ git clone https://github.com/LordOfTrident/telnexir
$ cd telnexir
$ mix run
```

### Client
```sh
$ telnet localhost 4040
```
